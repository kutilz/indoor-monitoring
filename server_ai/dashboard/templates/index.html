Kalibrasi Servo MG996R (Continuous Rotation)

Servo ini berputar terus selama diberi sinyal — tidak punya posisi sudut tetap. Tahan tombol CW/CCW untuk memutar pelan selama ditahan (kecepatan sesuai slider), lepas untuk berhenti. Tentukan Arah Klik per servo (arah putar yang menekan tombol AC, sesuai orientasi mounting masing-masing servo), lalu atur Durasi Klik (lama menekan) dan Durasi Kembali (lama putar balik ke posisi awal). Geser Trim Netral jika servo masih sedikit bergerak saat seharusnya diam.

🛟 Safety: jika koneksi ke node aktuator terputus saat tombol jog ditahan, atau jog berjalan >4 detik tanpa henti, servo berhenti otomatis. Tombol di bawah menghentikan SEMUA servo segera — gunakan jika ada gerakan tak terduga.
